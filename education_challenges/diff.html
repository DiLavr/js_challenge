Difference in angels

Task:



Write a diff function that takes two angles (integer), each from 0 to 360, and returns the smallest difference between them.

Example:

diff(0, 45) === 45; // not 315, but 45, because 45 is smaller
diff(0, 180) === 180;
diff(0, 190) === 170; // not 190, but 170, because 170 is smaller
diff(120, 280) === 160;
